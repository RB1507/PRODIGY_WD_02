# ⏱ Stopwatch App

A clean, minimal stopwatch web app with light/dark theme toggle.

## 🔧 Features
- Start / Pause / Reset functionality
- Lap time recording
- Keyboard shortcuts:
  - `A` → Start/Pause
  - `W` → Reset
  - `D` → Lap
- Light/Dark mode toggle with local storage support
